# Client Customization Plugin ## Descrição O Client Customization Plugin é um plugin simples para WordPress que adiciona uma mensagem personalizada ao final do conteúdo de posts. O objetivo é fornecer uma forma rápida de identificar a origem do conteúdo, adicionando uma nota informativa sobre o criador do site. ## Funcionalidades - Adiciona uma mensagem ao final de posts do tipo `post` no WordPress.
